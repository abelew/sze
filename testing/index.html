The following links may be used to visualize microarray data
for Arabidopsis thaliana.

One may view all of the data from a particular table.

<& selector.html, name => 'selectall_table', statement => qq"SHOW TABLES" &>

The pollen dataset has been partially annotated. Thus it
is possible to view expression data for a particular gene family.

<& selector.html, name => 'family', statement => qq"SELECT distinct(family) FROM pollen ORDER BY family" &>

The pollen dataset also has a defined set of tc codes.
These data may be graphed below.

<& selector.html, name => 'tc_code', statement => qq"SELECT distinct(tc_code) FROM pollen ORDER BY tc_code" &>

Placing Arabidopsis gene accessions in the following text-area
will graph the data from them specifically. Conversely, fill in some
keywords into the textbox and every accession with a corresponding
annotation will be graphed.
First choose a table to query.

<& selector.html, submit => undef, name => 'search_table', statement => qq"SHOW TABLES" &>
